# MallaClientes

Este proyecto contiene el dashboard y herramientas para la gestión de clientes.

## Estructura
- backend/
- business_rules/
- docs/
- frontend/
- PoyectoMallaCientes/
- proyecto_entregable/

Incluye archivos principales de frontend y backend, documentación y reglas de negocio.

## Instrucciones
1. Clona el repositorio.
2. Instala dependencias según los README de cada subcarpeta.
3. Sigue las instrucciones de cada módulo para ejecutar el proyecto.
