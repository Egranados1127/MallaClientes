# Backend FastAPI para MallaClientes

Este backend expone la lógica de scoring, segmentación y datos de clientes para el frontend React.

## Estructura sugerida
- main.py: punto de entrada FastAPI
- routers/: endpoints de negocio
- models/: modelos de datos
- services/: lógica de scoring y utilidades
- data/: archivos de datos (ejemplo: MallaClientes.xlsx)

Configura tu entorno virtual y dependencias con requirements.txt.
